# Kest Docs

Kest is an AI-native API testing platform built for developers who spend most of their time in the terminal with coding copilots.

It combines two surfaces:

- `Kest CLI`: a fast, markdown-native command line tool for testing and automation
- `Kest Web Console`: a collaborative dashboard for visualizing flows, results, and team activity

The documentation currently has two top-level sections:

- `Product Docs`: concepts, quickstart guidance, and day-to-day workflows
- `API Reference`: the future home for platform endpoints, auth details, and error semantics

If you are new to Kest, start with:

- `Product Introduction`
- `Quickstart`
- `AI-Native Workflow`
