# Quickstart

The content below is a minimal example used to verify code block rendering in the Kest docs site.

## 1. Install dependencies

```bash
pnpm install
pnpm dev
```

## 2. Initialize the client

```ts
import { Kest } from "@kest/sdk";

const client = new Kest({
  apiKey: process.env.KEST_API_KEY,
  baseUrl: "https://api.kest.dev/v1",
});

export async function runQuickstart() {
  const result = await client.models.generate({
    model: "kest-chat",
    input: "Hello from Kest Docs",
  });

  return result;
}
```

After rendering, the page should show language-aware code blocks with syntax highlighting.
