# Kest 文档

Kest 是一套面向 AI 协作时代的 API 测试平台，服务于长期在终端和编辑器中工作的开发者。

它由两部分组成：

- `Kest CLI`：高性能、Markdown 原生、适合自动化与脚本化的命令行工具
- `Kest Web Console`：用于查看测试流、历史结果和团队协作状态的可视化界面

当前文档分为两个一级栏目：

- `产品文档`：介绍 Kest 的核心理念、快速开始方式和开发工作流
- `API 参考`：后续会承载平台接口、鉴权和错误语义等内容

- `产品介绍`
- `快速开始`
- `AI 原生工作流`
