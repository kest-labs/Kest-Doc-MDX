# 快速开始

下面这段内容用于演示 Kest 文档站中的代码块渲染效果。

## 1. 安装依赖

```bash
pnpm install
pnpm dev
```

## 2. 初始化客户端

```ts
import { Kest } from "@kest/sdk";

const client = new Kest({
  apiKey: process.env.KEST_API_KEY,
  baseUrl: "https://api.kest.dev/v1",
});

export async function runQuickstart() {
  const result = await client.models.generate({
    model: "kest-chat",
    input: "Hello from Kest Docs",
  });

  return result;
}
```

运行后，你应该可以在页面上看到带语言标记的代码块，并由前端自动渲染高亮样式。
